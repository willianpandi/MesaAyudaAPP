# Mesa de Ayuda - CZ3

Poyecto generado con Angular CLI version 16.2.2.

## Descripción

API para la aplicación de Mesa de Ayuda, realizada para la Coordinación Zonal 3 - Salud.

## Instalación

```bash
$ npm install
```

## Build para produccion

```bash
$ ng build
```

## Correr la APP 
```bash
$ ng serve
```

# App Corriendo
En cualquier navegador `http://localhost:4200/`
